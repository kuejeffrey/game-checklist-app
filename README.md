# Level Up

A cozy daily survival and productivity tracker built with Flutter.

## Structure

- `lib/main.dart` - app entry point
- `lib/screens/` - top-level screens
- `lib/models/` - app data models
- `lib/services/` - local persistence
- `lib/utils/` - XP and leveling helpers
- `lib/widgets/` - reusable UI components

## Notes

The original imported files from the zip are preserved in `.codex_review_zip/`.
This workspace layout is organized so it is easier to open and work with in VS Code.
